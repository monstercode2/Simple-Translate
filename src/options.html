Simple Translate Options

Choose one translation provider, set your target language, then save. Ollama keeps requests on your machine; OpenAI-compatible APIs send text to the endpoint you configure.

1. Choose Provider

Use Ollama if you want local translation. Use OpenAI-compatible if you already have an API endpoint and model name.

2. Language

Ollama

Start Ollama locally before translating. A common setup is ollama pull qwen2.5:7b, then ollama serve.

OpenAI-compatible

Works with services that support /v1/chat/completions. Your API key is saved only in browser extension storage.
