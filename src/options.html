简单翻译

插件设置

选择一个翻译服务，设置目标语言，然后保存即可。Ollama 会把请求发到你的本机模型；OpenAI-compatible 会把文本发送到你配置的接口地址。

1. 选择翻译服务

想完全本地使用就选 Ollama；已经有 API Key、网关或第三方模型服务，就选 OpenAI-compatible API。

2. 语言设置

Ollama 本地模型

使用前请先启动 Ollama。常见命令：ollama pull qwen2.5:7b，然后运行 ollama serve。

OpenAI-compatible API

适用于兼容 /v1/chat/completions 的服务。API Key 只保存在浏览器扩展存储里，不会写入项目文件或 Git 仓库。
